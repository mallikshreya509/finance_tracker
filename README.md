# Finance Tracker
A simple backend application to track daily expenses, built using Spring Boot + MySQL.
This project demonstrates core backend concepts like REST APIs, database integration, and layered architecture.

 ## 🚀 Features
➕ Add new expense

📋 View all expenses

💾 Store data in MySQL database

🏗️ Layered architecture (Controller → Service → Repository)


## 🧱 Tech Stack
Java 17

Spring Boot

Spring Data JPA

MySQL

Maven

MongoDb
